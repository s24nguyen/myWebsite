AstroNOC

Project Description

AstroNOC was a project that I worked on during my internship at United Shore. The idea of this project was to create a web-based dashboard that would increase transparency for the entire IT department. Whenever a system was acting slow or something wasn't working, the NOC team would get a lot of calls/messages asking what is wrong. This wastes a lot of time time for the NOC team, so this application was created to allow users to quickly look up and see what the issue was themself.
